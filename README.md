# Optimal-Restoration-Planning-for-Low-Inertia-Distribution-Systems-Using-Grid-Forming-DERs
Supporting data for IEEE SEST 2026 paper on frequency-constrained restoration planning. Contains scenario generation data, investment cost parameters, virtual inertia coefficients, droop settings, and full nomenclature for the stochastic MILP model tested on the IEEE 33-bus distribution system.
